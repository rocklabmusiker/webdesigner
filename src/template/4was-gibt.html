Was gibt Ihrem Kind der Webdesign Kurs

1. 

Nach dem Webdesign Kurs wird sich Ihr Kind die wichtigsten Bereiche des Webdesigns beherrschen.

2. 

Ihr Kind erlernt die Auszeichnungssprache HTML und CSS von klein auf. Das ist das Fundament im Webdesign. Ohne funktioniert keine Webseite.

3. 

Ihr Kind lernt die neusten Trends und Internet Technologien kennen. Dabei entdeckt Ihr Kind neue Möglichkeiten die im Web vorhanden sind.

4. 

Schon nach 30 Unterrichtseinheiten wird Ihr Kind eine anspruchsvolle Internetseite erstellen können.
